# CLI Reference

This directory contains full argument and config details for each command family.

Use this index when you need exhaustive option-level behavior. For the shortest
onboarding path, use the top-level `README.md` quickstart first.

## Command Docs

- [`fit.md`](fit.md): `olmix fit` (canonical) and `olmix-fit` (legacy alias)
- [`generate.md`](generate.md): `olmix generate` and `GenerationConfig` fields
- [`launch.md`](launch.md): `olmix launch run|status|cancel|preview`
- [`priors.md`](priors.md): `olmix priors compute`

## Canonical Naming

- Canonical fit command: `olmix fit`
- Legacy alias: `olmix-fit`

The alias exists for compatibility with older scripts. Prefer `olmix fit` for
all new docs and automation.
